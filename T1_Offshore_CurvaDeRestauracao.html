Metodologia¶

No desenvolvimento deste trabalho iremos criar um mtodo de soluo para o problema de uma unidade flutuante que oscila conforme a variao a mar, considerando apenas o problema bidimensional, com a oscilao na direo near e far.

criada a rotina da catenria para variao do ngulo theta e da projeo horizontal total. Em cada iterao obtm-se um novo valor da fora horizontal pelo algoritmo de Newton-Raphson.

Calcula-se a curva de restaurao (Deslocamento Horizontal x Fora de Restaurao) para um riser isolado e para o sistema composto por dois risers para os pontos de um vetor delta x, que varia 10% da lmina d'gua em cada direo, em incrementos de 10 m.

O Problema¶

Cada linha pesa 0,2165kN/m na gua e possui um comprimento total de 2800m, sendo instaladas com trao horizontal de topo igual 400kN na posio de projeto.

A profundidade da gua igual a 500m e considera-se que as conexes das linhas com a unidade flutuante esto localizadas na superfcie da gua.

Pede-se:

a) Determinar a curva de restaurao de uma linha de ancoragem isolada.

b) Determinar a curva de restaurao do sistema de ancoragem composto pelas duas linhas.

Obs.: Considerar um offset mximo da UF igual a 10% da lmina dgua para os sentidos near e far. Apresentar as curvas na forma de tabela e grfico considerando incrementos de 10m at o offset mximo.

Utiliza-se a seguinte formulao para os parmetros do problema:

$$ a = \frac{F_x}{q}$$$$ x_s = a \cdot cosh^{-1} \biggl(\frac{y + a}{a}\biggr) $$$$ S = a \cdot senh\biggl(\frac{x_s}{a}\biggr) $$$$ W = q \cdot S $$$$ T = \sqrt{F_x^2 + W^2} $$$$ \theta = tg^{-1}\biggl(\frac{F_h}{W}\biggr) $$$$ x_h = S_t - S $$$$ x_t = x_h + x_s $$

Derivada Central¶

def df(f, x, h):
    """Derivada Central:
    Calcula uma aproximao da primeira derivada de f, nos pontos x, usando passos de tamanho h.
    :param f: Funo
    :type f: func
    :param x: Pontos
    :type x: float, int or array
    :param h: Passo
    :type h: float
    :return: Derivada de f no ponto x
    :rtype: float or array
    """
    return (f(x+h) - f(x-h))/(2*h)

Algorítimo de Newton-Raphson¶

Definiremos agora a funo que realizar o Algortimo de Newton-Raphson para encontrar as razes de uma funo:

def newton(f, x0, Y=0, prec_x=1e-6, prec_y=1e-6, maxsteps=100):
    """Algortimo de Newton-Raphson:
    Calcula o ponto x0 tal que f(x0)=Y
    :param f: Funo
    :type f: func
    :param x0: Ponto de Partida
    :type x0: float, int
    :param Y: Valor desejado da funo, defaults to 0 (Raz)
    :type Y: float, int
    :param prec_x, prec_y: Preciso nos valores de x e y, defaults to 1e-6
    :type prec_x, prec_y: float
    :param maxsteps: Nmero Mximo de Iteraoes, defaults to 100
    :type maxsteps: int
    :return: Lista dos valores de x calculados
    :rtype: list
    """
    x_list = []
    for i in range(maxsteps):
        passo = (f(x0)-Y)/df(f, x0, prec_x)
        x = x0 - passo
        x_list.append(x)
        if abs(passo) < prec_x or abs(f(x)-Y) < prec_y:
            return x_list
        x0 = x
    raise "O Nmero Mximo de Iteraes foi atingido"

Definiremos agora a funo que realizar as rotinas necessrias para resolver o problema da catenria.

variado o ngulo Theta e o comprimento total da projeo horizontal. Dentro desse loop calculada a fora horizontal com Newton-Raphson.

Para chutes iniciais de Theta e Xt foram adotados valores pequenos que permitem a convergncia. O algoritmo encontra indeterminao se esse valor for determinado como '0' (zero).

def catenaria(St, Xt, y, q, theta, prec=1e-3):
    """Catenria com rotina de Newton-Raphson:
    Soluciona o problema de Riser apoiado por cabo
    :param St: Comprimento Total do Riser (m)
    :type St: float, int
    :param xt, y: Distncia entre os apoios (m)
    :type xt, y: float, int
    :param q: Peso por unidade de comprimento (kN/m)
    :type q: float, int
    :param theta: Chute inicial do ngulo entre a resultante e a vertical ()
    :type theta: float, int
    :param prec: Preciso do Resultado, defaults to 1e-3
    :type prec: float
    :return: Fora Horizontal no apoio(kN); ngulo entre a resultante e a vertical(rad); Fora Total no apoio(kN)
    :rtype: float
    """
    # chutes iniciais de theta e Fx
    theta_i = 1
    theta_f = theta
    Fx_i = 1
    # definindo Xt_i somente para passar no primeiro while
    Xt_i = 1
    # primeiro loop, eStimando y e Xt2 a partir de um valor arbitrado de theta
    while abs(theta_f-theta_i) > prec:
        theta_i = theta_f
        y_i = y
        Xt_2 = Xt
        # segundo loop, estimando xs, s, W, theta, xh e Xt1 a partir de um valor arbitrado de Fx
        while abs(Xt_i-Xt_2) > prec:
            a = Fx_i/q
            xs = a*np.arccosh((y_i+a)/a)
            s = a*np.sinh(xs/a)
            W = q*s
            T = np.sqrt(Fx_i**2 + W**2)
            xh = St-s
            # trocando theta para prxima iterao
            theta_f = np.arctan(Fx_i/W)
            # trocando Xt para prxima iterao
            Xt_i = xh+xs
            # definindo a funo G(Fx)=Xt
            def G(Fx): return (xh + ((Fx/q)*np.arccosh((y_i*q+Fx)/Fx)))
            # refazendo o processo utilizando newton-raphson para obter o novo valor de Fx
            lista = newton(G, x0=Fx_i, Y=Xt_2)
            Fx_i = lista[-1]
    return Fx_i, theta_f, T

Conclusões¶

Quanto curva de restaurao para a linha isolada, possvel observar que a curva se comporta praticamente exponencialmente conforme o a unidade flutuante se desloca. Quanto mais distante da posio de equilbrio, maior a fora restauradora, uma vez que maior a trao na linha, e a componente horizontal aumenta buscando voltar posio inicial. Analisando uma linha isolada, se a unidade flutuante se deslocar para outro sentido, essa linha ir "relaxar", diminuindo a componente horizontal da fora que busca o equilbrio.

A geometria, para um offset de 10% da lmina d'gua nos sentidos "near" e "far" se altera. Observa-se a distncia horizontal em cada posio sendo:

No equilbrio:

xs = 1330.330m

Xt = 2682.029m

Para dx = +50m

xh = 1401.699m

xs = 1330.330m

Xt = 2732.029m

Para dx = -50m

xh = 1301.699m

xs = 1330.330m

Xt = 2632.029m

Sendo xh a distncia do riser em solo, xs a projeo horizontal do riser suspenso e Xt o comprimento horizontal total do riser ao TDP.

J considerando o sistema de ancoragem, considerando a fora de restaurao para voltar ao equilbrio como a diferena das foras nas duas direes, nas duas linhas, temos que o valor da fora restauradora diminui, uma vez que enquanto uma linha trabalha para retornar ao equilbrio, a outra possui componente horizontal no sentido contrrio. Ento, a fora que se tem a diferena entre os valores +dx e -dx.

O estudo das foras restauradoras se mostram de grande importncia para que na unidade flutuante, ao atingir uma determinada distncia, saiba qual fora deve ser aplicada na linha para manter a plataforma em equilbrio
